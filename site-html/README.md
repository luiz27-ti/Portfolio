Projeto Web – HTML, CSS e JavaScript

Site desenvolvido como projeto avaliativo da AV2 (2024/2).

Requisitos atendidos:
- Múltiplas páginas HTML com navegação
- Cabeçalho e rodapé em todas as páginas
- Estrutura de pastas organizada
- CSS próprio (inline, interno e externo)
- Interações com JavaScript (sem uso de alert)
- Tabela
- Barra lateral
- Uso de imagens
- Uso de vídeo ou áudio

Tecnologias utilizadas:
- HTML5
- CSS3
- JavaScript puro

Projeto desenvolvido individualmente, sem uso de frameworks ou templates.
